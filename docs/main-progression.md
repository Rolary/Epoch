# 《生态纪元》第一版主线推进

本文档用于锁定第一版线性主线，防止开发时把游戏写成无主次的控制面板。第一版只做“生命诞生篇”。

## 1. 第一版主题

```text
生命诞生篇：让第一批生命从始源潮池中出现。
```

玩家应该感受到的主线不是“攒资源买升级”，而是：

```text
潮池富集材料
-> 分子结构留下痕迹
-> 复制链出现
-> 反应被膜泡包裹
-> 原初细胞诞生
-> 生命开始追逐光照
```

## 2. 主线阶段总览

| 阶段 | 玩家理解 | 核心目标 | 新内容 | 代表反馈 |
| --- | --- | --- | --- | --- |
| 0. 创建生态 | 为这颗星球命名，选择长期倾向 | 创建存档并选择源质印记 | 生态名称、源质印记 | 潮池被唤醒 |
| 1. 富集材料 | 潮池需要生命材料 | 有机质达到首个目标 | 主操作、少量资源 | 有机分子开始聚集 |
| 2. 留下痕迹 | 分子结构开始稳定存在 | 点亮有机富集 | 演化入口 | 矿物晶面留下反应痕迹 |
| 3. 学会复制 | 生命开始具备延续能力 | 解锁自复制链 | 突变点、谱系准备 | 异常链体反复出现 |
| 4. 包成细胞 | 反应被膜泡保护 | 解锁原始膜泡或原初细胞 | 第一批物种、图鉴 | 第一条谱系被记录 |
| 5. 追逐光照 | 生命走向能量利用 | 接近感光色素或光合作用前夜 | 生态事件、遗产可能性 | 潮池出现蓝绿色微光 |

## 3. 阶段 0：创建生态

### 目标

让玩家建立“这是我的星球生命史”的身份感。

### 玩家看到

- 生态名称输入。
- 3 个随机源质印记。
- 每个印记包含图标、等阶、名称、短描述。
- 一个明确的进入按钮。

### 玩家操作

```text
输入生态名称
-> 选择 1 个源质印记
-> 创建生态
```

### 解锁内容

- 始源潮池首页。
- 当前目标“富集材料”。
- 主操作“催化潮池”。

### 禁止

- 不展示注册、密码、手机号等正式账号流程。
- 不展示全量系统入口。
- 不展示完整演化树。

## 4. 阶段 1：富集材料

### 玩家理解

潮池还没有生命，只有能量、有机质和矿物在积累。

### 核心目标

```text
有机质达到首个目标值。
```

### 玩家看到

- 当前目标卡：富集材料。
- 始源潮池主场景。
- 主操作：催化。
- 关键资源：有机质、能量。
- 最近反馈。

### 玩家操作

- 点击潮池或主操作按钮。
- 观察资源和目标进度增长。

### 代表反馈

```text
温热浅水中漂浮着新的有机分子。
矿物晶面捕获了一批反应材料。
闪光让潮池短暂升温。
```

### 推进条件

- 有机质达到目标。
- 玩家至少完成数次有效催化。

### 解锁内容

- 演化入口。
- 第一个演化节点“有机富集”。

## 5. 阶段 2：留下痕迹

### 玩家理解

一些分子结构不再只是随机出现，而是开始在潮池中稳定留下痕迹。

### 核心目标

```text
点亮有机富集。
```

### 玩家看到

- 演化入口出现。
- 演化页只展示早期路径，不展示完整未来科技树。
- 首页目标切换为“点亮有机富集”。

### 玩家操作

- 进入演化页。
- 点亮可解锁节点。
- 回到潮池继续催化。

### 代表反馈

```text
有机结构开始在潮汐边缘反复出现。
潮池不再只是混沌反应，它开始留下模式。
```

### 推进条件

- 有机富集已解锁。
- 矿物质或稳定性达到下个节点需求。

### 解锁内容

- 策略操作的入口或底部半屏选择。
- 资源展示扩展到矿物质、稳定性。
- 下个主线目标“推动自复制链”。

## 6. 阶段 3：学会复制

### 玩家理解

生命真正的起点不是材料变多，而是某些结构能把自己延续下去。

### 核心目标

```text
解锁自复制链。
```

### 玩家看到

- 演化路径推进到自复制链。
- 稳定性和突变倾向开始变得重要。
- 策略操作开始带来短期收益和长期代价。

### 玩家操作

- 选择潮汐扰动、矿物沉积、提高温度等策略操作。
- 在资源增长、稳定性和突变之间做取舍。

### 代表反馈

```text
一段链体在断裂后留下了相似结构。
复制错误没有立刻消失，反而变成了新的可能。
```

### 推进条件

- 自复制链节点解锁。
- 至少触发一次与复制、突变或稳定窗口相关的事件。

### 解锁内容

- 突变点资源。
- 第一批谱系生成条件。
- 后续源质印记待选择条件之一。

## 7. 阶段 4：包成细胞

### 玩家理解

生命需要边界。膜泡把反应包裹起来，原初细胞开始出现。

### 核心目标

```text
解锁原始膜泡或原初细胞，并记录第一条谱系。
```

### 玩家看到

- 目标卡提示“制造稳定窗口，等待第一条谱系出现”。
- 第一个物种发现弹层。
- 图鉴入口解锁。
- 首页出现最近发现卡片。

### 玩家操作

- 继续催化和选择策略操作。
- 查看第一条物种图鉴。
- 理解物种的生态角色、生态位、影响和脆弱点。

### 代表反馈

```text
薄膜包住了一小段持续反应。
系统记录到第一条稳定谱系。
```

### 推进条件

- 原始膜泡或原初细胞节点解锁。
- 至少生成 1 个早期物种。

### 解锁内容

- 图鉴入口。
- 物种详情。
- 生物量资源。
- 物种生态影响。

## 8. 阶段 5：追逐光照

### 玩家理解

第一批生命开始利用外部能量，潮池即将进入更复杂的生态阶段。

### 核心目标

```text
接近感光色素或光合作用前夜。
```

### 玩家看到

- 潮池视觉出现蓝绿色微光或光照变化。
- 新事件与生态压力出现。
- 物种可能繁盛、濒危、灭绝或化石化。
- 第一次遗产形成后，遗产入口解锁。

### 玩家操作

- 选择是否提高光照、保护谱系或承受生态压力。
- 查看遗产或日志。

### 代表反馈

```text
一些原初细胞开始向光亮处聚集。
光照带来能量，也带来新的选择压力。
一条早期谱系退出了现存生态，却留下了可追溯的遗产。
```

### 第一版结尾体验

第一版可以停在“光合作用前夜”，给玩家明确预告：

```text
潮池中出现了第一缕蓝绿色微光。下一场生态爆发即将改变整颗星球的大气。
```

## 9. 主线推进禁止事项

- 禁止跳过“富集材料”直接展示物种。
- 禁止开局直接展示中后期资源。
- 禁止把全部演化节点一次性铺满。
- 禁止把策略操作做成无代价的收益按钮。
- 禁止让玩家只看数字，不知道生命史发生了什么。
- 禁止物种出现后只写日志、不改变生态。

## 10. 验收标准

- 玩家 3 秒内知道第一步要点哪里。
- 玩家 2 到 3 分钟内看到演化入口。
- 玩家 5 到 10 分钟内理解“材料 -> 结构 -> 复制”的主线。
- 玩家 10 到 20 分钟内至少发现 1 个物种。
- 玩家能说出当前阶段的生命史意义，而不是只说“我在攒资源”。
